Project Idea

Overview:
Creating a mobile app that can act as a controller for consoles like the PlayStation 4 or XBOX. This application can be used wireless from the console and will have the same functions as a controller would. There will have to be a link between the phone and the console, so maybe the same application will have to be downloaded onto the console. The app is free of charge.

Motivation:
If someone is looking for a spare controller or do not have the cash to afford one, this application is the solution to those problems. An application that has access to multiple types of controllers will be useful. You will never need to buy another controller ever again. This application is all you will need to use as a controller for any console.

Description:
The application will be able to hold multiple types of controllers. For specific games such as Tekken or Street Fighter, an arcade stick can be added as one of the controllers in the app. When buttons are pressed there can be vibration effects and these vibrations should have an option to be disabled. Adjustments of the user interface can also be another feature for the application so users can have their preferred layout. The size and location of the controls on screen should be an option for users to adjust. Sound effects are not necessary for this application unless it is wanted. A list of the types of controllers should also be in the application.

Tools and Technologies:
A laptop or desktop is needed to use software and write code for the application. Software such as Android Studio can be used to create apps.

Skills:
To make an app, knowledge of Java is needed. The software to make an app can be found on the internet. Software such as Android studio can be used to create apps. Knowledge about Java can also be accessed on the internet. A major skill for this app will be the design of each of the controllers. Database skills will also be needed to store the types of controllers.

Outcome:
If this project becomes successful, everyone with a phone will be able to use every type of controller for free through this application. Although this app is free, it may cause conflict in the market because real controllers cost cash.
